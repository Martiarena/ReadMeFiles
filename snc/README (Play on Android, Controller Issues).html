Upgrade to this version without losing progress or unlockables

Simply replace the new "Saves" folder with the "Saves" folder from the previous version, you will keep not just the progress and unlockables but also the score and control settings.

Controller Issues

If you have problems with the control settings just delete the "Saves" folder (or just delete the "Silver_Night's_Crusaders_(E).cfg" file).

The OpenBOR configuration will go to default, so you can configure everything again.

Here is a video tutorial on how to setup the gamepads, it works for all OpenBOR games created by True Role Dreams.

Possible incompatibility with Xbox controllers

This game runs on OpenBOR, a freeware software developed by Senile Team, and I think the developers still have problems with certain peripherals. So I recommend using a PC joystick or keyboard to play, those are the devices which this game was tested.
