Upgrade to this version without losing progress or unlockables

Simply replace the new "Saves" folder with the "Saves" folder from the previous version, you will keep not just the progress and unlockables but also the score and control settings.

Controller Issues

If you have problems with the control settings just delete the "Saves" folder (or just delete the "Neon_Lightning_Force_(E).cfg" file).

The OpenBOR configuration will go to default, so you can configure everything again.

Possible incompatibility with Xbox controllers

This game runs on OpenBOR, a freeware software developed by Senile Team, and I think the developers still have problems with certain peripherals. So I recommend using a PC joystick or keyboard to play, those are the devices which this game was tested.

Glitches List

Bug List

  • Thundervolt's feather:
    I use some unestable code to summon Steven's pet xD, the players who gets the item "Thundervolt's feather" can act "strange" for a few seconds. I recomend to wait a little until the eagle flyes from the ground, unless you want that your character get freeze. It can recover the movement if you wait but I don't know if it will freeze forever x.x, in the case that happen you can wait until the time over that kill your character and let you recover the control over him.
  • 2 Players bike bonus game:
    Can shut down the game if a second player gets into the game or use a credit, DURING the bike bonus stage. There's no glitch if both players gets into the game before or after the bike bonus stage.
